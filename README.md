# Racing Game Demo

This repository contains the source code for a small racing game demo created with **Construct 2**. The goal of the game is to drive the car as fast as possible while staying on the road.  

The project is exported as **HTML5**, making it playable directly on GitHub Pages. This demo is part of my portfolio and showcases my skills in **game development with Construct 2**.

## Features
- Simple, fun racing mechanics
- Road boundaries to avoid
- HTML5 export, playable in a browser
- Designed as a portfolio project to demonstrate game development skills

## Technologies
- Construct 2
- HTML5
